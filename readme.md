# Xray VLESS + XTLS Reality (Multi-User)

Это полнофункциональный, self-hosted VPN для обхода блокировок. Содержит только самое необходимое: Xray Core, VLESS, XTLS Reality протокол, и bash-скрипты для управления доступами. Проект создан в парадигме Docker Compose без сложных зависимостей.

## Главные фичи
* **Xray VLESS+Reality** (работает через 443 порт).
* **Multi-User**: У каждого клиента свой UUID.
* **Статистика**: Xray API снимает трафик для каждого пользователя отдельно!
* **Управление скриптами**: Без тяжелых веб-админок.

---

## 1. Подготовка Сервера (Ubuntu 22.04)

1. Арендуйте VPS на Hetzner (например CX23).
2. Зайдите по SSH и установите требования:
   ```bash
   sudo apt update
   sudo apt install -y docker.io docker-compose-v2 jq openssl curl
   ```
3. Откройте нужные порты в `ufw`:
   ```bash
   sudo ufw allow 22/tcp
   sudo ufw allow 443/tcp
   sudo ufw enable
   ```
4. Склонируйте этот репозиторий в `/root/vpn`.

---

## 2. Инициализация и Запуск

Перейдите в папку проекта:
```bash
cd /root/vpn
```

Запустите команду, которая сгенерирует ключи, `.env`, пустую базу пользователей `users.json`, и соберет итоговый конфиг Xray:
```bash
make init
```

Запустите сервер в фоне:
```bash
make up
```

---

## 3. Управление Пользователями

### Добавить пользователя
Запуск команды создаст новый уникальный UUID и сгенерирует ссылку:
```bash
make add-user USER=alice
```
В консоли вы получите VLESS Reality ссылку, начинающуюся на `vless://...`.

### Посмотреть статистику пользователей
Узнайте, кто и сколько тратит трафика (Uplink, Downlink) в человекочитаемом виде (KB/MB/GB):
```bash
make stats
```

### Вывести список доступов
```bash
make list-users
```
Покажет всех зарегистрированных (активных и отозванных).

### Отозвать доступ
Блокирует пользователя (отключает от Xray), не удаляя его историю из системной статистики:
```bash
make revoke-user USER=alice
```

### Запросить конфиг снова
Хотите вывести QR (через сторонние CLI утилиты) или просто VLESS-строку для уже существующего пользователя?
```bash
make client-config USER=alice
```

---

## 4. Клиенты (NekoBox & Shadowrocket)

Этот сервер не имеет клиентского приложения, он совместим с лучшими клиентами:
* **Android**: Скачайте *NekoBox for Android* из GitHub/Google Play. Нажмите + и выберите "Import from Clipboard" после копирования вашей `vless://` ссылки.
* **iOS**: Купите *Shadowrocket* в App Store. Нажмите + (Type: Vless) или просто отсканируйте QR код, переведенный из этой же ссылки.

---

## 5. Обновление (Troubleshooting)

Если вы сломали ключи или всё перестало работать:
1. Выполните проверку системы: `make healthcheck`
2. Пересоберите конфиги в ручную: `make render`
3. Сбросьте ключи: `make regenerate-secrets`

**Логи сервера**:
```bash
make logs
```
